# lottery-capybara

卡比巴拉抽獎大會是一個純前端、零相依的抽獎頁面。它用靜態 HTML、CSS、JavaScript 實作，可直接用瀏覽器開啟，不需要安裝 Node 套件、建置工具或後端服務。

## 功能

- 可調整抽出人數
- 可點擊卡片取消或恢復參與資格
- 抽獎時有逐張掃描動畫
- 支援一次抽出多位得獎者
- 得獎後會顯示排行榜與彩帶效果
- 重置會恢復完整初始狀態

## 使用方式

### 直接開啟

最簡單的方式是直接用瀏覽器開啟 [index.html](index.html)。

### 本機靜態伺服器

如果你想用本機 server 開啟，也可以在專案目錄執行：

```bash
python3 -m http.server 8000
```

然後在瀏覽器打開：

```text
http://localhost:8000
```

## 操作說明

1. 點擊卡片可切換該角色是否參與抽獎。
2. 用加減按鈕調整抽出人數。
3. 點擊「開始抽獎！」執行抽獎。
4. 得獎視窗中的「再抽一次」只會關閉本輪結果，保留目前參與名單。
5. 「重置」會把抽出人數恢復為 1，並讓所有角色重新參與。

## 專案結構

- [index.html](index.html): 頁面結構與 DOM 入口
- [style.css](style.css): 頁面樣式、動畫、卡片與彈窗視覺
- [script.js](script.js): 抽獎邏輯、狀態管理、彩帶動畫與互動事件

## 技術說明

- 無框架、無套件管理器
- 抽獎邏輯使用 Fisher-Yates 洗牌，避免用 `sort(() => Math.random() - 0.5)` 帶來的偏差
- 可直接部署到任何靜態站台，例如 GitHub Pages、Netlify 或一般 Nginx 靜態目錄
